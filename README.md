# Workly SaaS

Gestor de tareas interno para equipos de desarrollo de software.

## Descripción

Este proyecto es un sistema de gestión de tareas diseñado para mejorar la asignación, el seguimiento y la actualización de las tareas internas dentro de una empresa de desarrollo de software. La aplicación web facilitará una gestión eficiente y trazable de las tareas.

## Tecnologías

- Next.js
- TypeScript
- Tailwind CSS
